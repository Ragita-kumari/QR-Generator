📱 QR Code Generator
A sleek, responsive web application that converts URLs or text into high-quality QR codes instantly. Built with a focus on clean UI and smooth user experience.

✨ Features
- Instant Generation: Uses the QRServer API to generate codes in real-time.

- Responsive Design: Fully optimized for mobile, tablet, and desktop screens.

- Dynamic UI: The interface expands gracefully when the QR code is generated.

- Dark Mode Ready: Includes CSS classes prepared for a dark-themed user experience.

- User Feedback: Features loading states on the button to inform the user during the API call.

- Input Validation: Prevents redundant API calls if the input remains the same or is empty.


🚀 Demo
- Enter a URL or any text in the input field.

- Click the Generate QR Code button.

- The container will expand to reveal your custom QR code.

🛠️ Built With
- HTML5: Semantic structure.

- CSS3: Custom styling with Flexbox and transitions for smooth animations.

- JavaScript (ES6): DOM manipulation and asynchronous image loading.

- API: goqr.me for QR code generation.

📂 Project Structure
├── index.html   # Main structure
├── style.css    # Custom styling and animations
└── script.js    # Logic for API fetching and UI toggling

⚙️ How to Run
- Clone the repository:
git clone https://github.com/your-username/qr-code-generator.git

- Navigate to the folder:
cd qr-code-generator

- Open the project:
Simply open index.html in your favorite web browser

📝 Usage Note
The generator relies on an active internet connection to fetch the QR code image from the API.
